🔌 Редактор плат SV005 / SV004 (ручное назначение)

SV005: 0 SV004: 0 Всего: 0
▼ Для SV005: два плинта (R1/D1 и R2/D2). Для SV004: один плинт с 4 пинами (0–3), на каждом пине можно выбрать до двух устройств (ИК, КАВ, СМК, ДРС) с указанием количества (по умолчанию 1).
• Нумерация плинтов сквозная по порядку плат: SV005 даёт 2 плинта, SV004 – 1 плинт.
• У каждого пина SV004 можно указать свой номер помещения (если не указан, ячейка остаётся пустой).
• Кнопка "Копировать вход → выход" дублирует настройки (только для SV005).
