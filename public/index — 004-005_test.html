🔌 Редактор плат SV005 / SV004 (ручное назначение)

▼ Для SV005: два плинта (R1/D1 и R2/D2). Для SV004: один плинт с 10 пинами (0–9).
• Нумерация плинтов сквозная по порядку плат: SV005 даёт 2 плинта, SV004 – 1 плинт.
• Выберите устройство и введите номер кабеля для каждого пина.
• Для SV004 доступны: ИК, КАВ, СМК-ДРС, ИК-СМК.
• Кнопка "Копировать вход → выход" дублирует настройки (только для SV005).
